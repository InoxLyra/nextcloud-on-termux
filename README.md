## About

An easy-to-use script for installing nextcloud on termux.

## References

- [Udocker](https://github.com/indigo-dc/udocker)
- [Termux](https://github.com/termux/termux-app)
- [Nextcloud](https://github.com/nextcloud/server)

## Installation

1. **Download the script**

```bash
curl -O https://raw.githubusercontent.com/InoxLyra/nextcloud-on-termux/refs/heads/main/SetupNextcloud.sh
```

2. **Grant execution permission**

```bash
chmod +x SetupNextcloud.sh
```

3. **Run the script**

```bash
./SetupNextcloud.sh
```

## Troubleshooting

If your device screen turns off during script execution, you may need to enable the "Keep screen on" option in Termux. This ensures that the screen remains on while the script is running.
